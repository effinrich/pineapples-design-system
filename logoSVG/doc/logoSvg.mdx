# LogoSVG

LogoSVG is for easily managing and resizing the Vatom logo.

## Usage

```tsx
import * as React from 'react';
import { LogoSVG } from '@pineapples/ui/LogoSVG';

const MyComponent = () => <LogoSVG height="50px" width="20%" />;
```
